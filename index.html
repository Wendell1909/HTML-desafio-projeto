Tag p ⬆️

A tag p é usada para criar parágrafos de texto.

< p >Este é um parágrafo de texto< /p >

Tag mark ⬆️

A tag mark é usada para destacar texto, geralmente com uma cor de fundo amarela.

Tag strong ⬆️

A tag strong é usada para destacar o texto com ênfase forte, geralmente exibido em negrito.

Tags ol, ul, li ⬆️

As tags ol e ul são usadas para criar listas ordenadas e não ordenadas, respectivamente. A tag li define um item da lista, quantos li tiverem sera quantos itens a lista vai ter.

  1. Item 1
  2. Item 2
  • Item A
  • Item B

Tag a ⬆️

A tag a é usada para criar links. O atributo href define o destino do link.

< a href = "http://www.google.com">Clique aqui< /a >

Tag hr ⬆️

A tag hr é usada para inserir uma linha horizontal para separar conteúdos.
